# Another Tower Defense - Some French Devs

Another Tower Defense is a small 2D tower defense game made for **Mini Jam 136** in 72 hours by our team, **Some French Devs**.
You build towers and fields to survive waves of enemies on a pixel-art battlefield.

## Game jam

- **Jam**: Mini Jam 136 (hosted by ZahranW).
- **Duration**: 72 hours.
- **Team**: Some French Devs (first game jam).

## Controls

- **Left click**: Select a tower or a field.
- **Right click** or **Escape**: Deselect the current selection.
- **Space**: Speed up the game.

## Platforms

- **Windows** (tested on Windows 10+).

## How to play (itch.io build)

1. Go to the [itch.io page of the project](https://some-french-devs.itch.io/another-tower-defense)
2. Click on the "download executable" button ( will download **Another Tower Defense.zip**)
3. Extract it
4. Run the executable
5. Enjoy !

Because the game is a packed executable, some antivirus tools may flag it; if you prefer, you can run it from source instead.

## How to run from source

You need **Python 3.10+** with `pygame-ce` and `pytmx` installed.

1. Install Python 3.10+ from the official website.  
2. Install dependencies:

```bash
python -m pip install pygame-ce pytmx --upgrade
```

3. Clone this repository:

```bash
git clone https://github.com/Spishewi/minijam-136-somefrenchdevs.git
cd minijam-136-somefrenchdevs
```

4. Run the game:

```bash
python main.py
```
